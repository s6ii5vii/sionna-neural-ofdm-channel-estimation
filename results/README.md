# Results

Experiment runners save plots to `results/figures/` and machine-readable tables
to `results/tables/`.

Generated results should be reproducible from a committed configuration and
experiment script. Avoid committing large sweeps, model checkpoints, or
intermediate arrays. Small, meaningful tables or figures may be committed later
when they support a reviewed report and their provenance is documented.

The repository does not currently ship claimed neural-performance results.
